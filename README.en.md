#  bsin-server-waas

#### Description
火源钱包是一款采用先进的前后端分离和云原生微服务架构设计的企业级MPC钱包。以WaaS（Wallet as a Service）为核心理念，旨在为企业提供强大的Web3资产管理能力，确保资产的安全控制和流畅转移。该钱包不仅支持多链管理和高级加密算法，还优化了用户体验和操作便捷性，为企业用户和个人投资者提供了一体化、安全可靠的数字资产管理解决方案。
